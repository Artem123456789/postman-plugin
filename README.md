# postman-plugin

<!-- Plugin description -->
This plugin allows to use postman inside IDE window

<!-- Plugin description end -->

## Installation

- Using the IDE built-in plugin system:
  
  <kbd>Settings/Preferences</kbd> > <kbd>Plugins</kbd> > <kbd>Marketplace</kbd> > <kbd>Search for "PostmanAgent"</kbd> >
  <kbd>Install</kbd>
